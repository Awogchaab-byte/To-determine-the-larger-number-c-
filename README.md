# 🔢 Integer Comparison Using Ternary Operator (C++)

This C++ program reads two integers from the user and determines which one is larger using the **conditional (ternary) operator**.

---

## 🧠 Objective
To practice input/output handling and the use of the conditional operator in C++.

---

## 📜 Problem Statement
